# RP2040-PICO-PC
Board of peripherals for Raspberry Pi RP2040-PICO which adds:

- HDMI/DVI 
- Audio 
- Micro SD card 
- Li-po charger 
- UEXT
- USB-OTG 

Allows to create a small PC with monitor and keyboard.

The RP2040-PICO-PC design is Open-Source Hardware (OSHW).
